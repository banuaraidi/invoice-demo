
# Invoice Demo

### Requirement
1. PHP 7
2. Nodejs 10
3. Installed composer


### Step by step to run

1. run composer install
2. copy file .env.example to .env
1. configure database connection on .env file
1. run `php artisan key:generate`
1. run `php artisan migrate`
1. run `npm install`
1. run `npm run dev`
